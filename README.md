### 풀이 방식
1. **코드 파일**은 '사이트_난이도_문제 이름.java' 형식으로 저장.
    - 예시: 'BOJ_12345_홍길동.java', 'PG_Lv2_홍길동.java'
2. **문제 풀이**는 블로그에 작성, 코드는 링크에 추가.
3. **폴더 별 README.md**에 문제 목록과 개념 정리.
